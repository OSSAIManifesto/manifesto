# Contributing

Thanks for your interest in the Open Source AI Manifesto! This project belongs to the community, and we welcome contributions of all kinds.

## Ways to contribute

- **Improve the website** — fix bugs, enhance accessibility, add animations, or improve responsiveness.
- **Add badge formats** — new embed formats or badge variants.
- **Fix typos and improve copy** — clarity matters.
- **Propose new principles** — see below.

## Proposing new principles

Changes to the manifesto's principles are high-impact and should be discussed before implementation. Please **open an issue first** describing:

- The principle you'd like to add or change
- Why it matters for the open source AI community
- How it relates to the existing principles

Do not open a PR with principle changes without a corresponding issue that has been discussed and approved.

## Getting started

### Prerequisites

- [Node.js](https://nodejs.org) (v18 or later)
- npm

### Setup

```bash
git clone https://github.com/OSSAIManifesto/manifesto.git
cd manifesto
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally. The dev server supports hot reloading.

### Build

```bash
npm run build
```

This generates a static export. Make sure the build passes before submitting a PR.

## Submitting a pull request

1. Fork the repository and create a branch from `main`.
2. Make your changes.
3. Verify the site builds without errors (`npm run build`).
4. Test your changes locally.
5. Open a pull request with a clear description of what and why.

## Code of conduct

Be respectful, constructive, and kind. This project is about sustaining trust in open source — let's lead by example.
